HTML e CSS

HTML e CSS são linguagens de desenvolvimento, não podemos chama-las de linguagens de programação, mas podemos falar que estamos desenvolvendo em linguagem HTML e CSS.

O que é HTML e CSS?

HTML - é uma linguagem voltada para conteúdo:

HYPERTEXT

MARKUP

LANGUAGE

CSS - é uma linguagem voltada para desing:

CASCADING

STYLE

SHEETS

Objetivos principais das linguagens?

HTML: textos, imagens, vídeos, tabelas...

CSS: cores, sombras, tamanhos, posicionamentos...
